# Drolosoft Homebrew Tap

Homebrew formulae for [Drolosoft](https://drolosoft.com) tools.

## Install

```sh
brew tap drolosoft/tap
brew install cmux-resurrect
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `cmux-resurrect` | Session persistence for cmux — save, restore, and manage your terminal workspaces |
